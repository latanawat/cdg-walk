⚙️ ตั้งค่า Firebase

แอปนี้ใช้ Firebase เป็น backend กรุณาตั้งค่าก่อนใช้งาน

1
ไปที่ console.firebase.google.com → สร้าง Project ใหม่
2
เปิด Firestore Database → Start in Test Mode
3
เปิด Storage → Start in Test Mode
4
Project Settings → Web App → Copy firebaseConfig
5
เปิดไฟล์ index.html แล้ว แทนที่ส่วน FIREBASE CONFIG HERE ด้วย config ของคุณ
// ตัวอย่าง config:
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "myapp.firebaseapp.com",
  projectId: "myapp",
  storageBucket: "myapp.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456:web:abc..."
};
